To Install the `canvafy` module, open a console and write the code below.
<br> </br>

[![NPM](https://nodei.co/npm/canvafy.png?mini=true)](https://discord.gg/luppux)

For **npm**
```console
npm i canvafy
```

For **Yarn**
```console
yarn add canvafy
```
